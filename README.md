# 🐾 Prototipo Tamagotchi Pet

Um projeto criado para estudar desenvolvimento de jogos, arquitetura de software e programação em C++ utilizando SFML.

---

# 📖 Sobre o Projeto

Este projeto foi criado com o objetivo de estudar e praticar conceitos importantes de desenvolvimento de software e criação de jogos, enquanto desenvolvo um jogo funcional durante o processo.

O projeto servirá como base para futuros projetos similares e para aprofundamento em arquitetura de jogos, organização de código e Programação Orientada a Objetos.

O jogo é inspirado no clássico jogo mobile **POU**, porém com uma proposta diferente:

> Um pet virtual que deve ser cuidado simultaneamente por duas pessoas.

A ideia principal é criar uma experiência interativa voltada para casais, onde ambos os jogadores compartilham o cuidado e evolução do pet.

# 🎮 Ideia do Jogo

O jogo consiste em um pet virtual compartilhado entre duas pessoas.

Os jogadores poderão:

* Alimentar o pet
* Brincar com o pet
* Cuidar de necessidades
* Evoluir o pet
* Desenvolver personalidade e emoções
* Compartilhar progresso

O objetivo é transformar o pet em uma experiência emocional e interativa.
